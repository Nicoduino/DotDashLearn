# DotDashLearn - User Guide

Welcome to DotDashLearn! This guide will help you get the most out of your Morse code learning experience.

## Getting Started

### System Requirements
- **Browser**: Chrome, Firefox, Edge, Safari (latest versions)
- **Device**: Desktop, Laptop, Tablet, or Smartphone
- **Internet**: Required only for initial setup; works offline
- **Audio**: Speakers or headphones

1. **Navigate to the application**

2. **Explore the interface**

3. **Enable audio**

## Learning Methods Explained

### Method 1: Practice Mode - Active Training

**What it does**: Tests your ability to recognize Morse code by ear

**Best for**: 
- Quick daily practice sessions (5-15 minutes)
- Reinforcing recently learned characters
- Improving recognition speed

**How to use**:
1. Click "Practice" on the home page or navbar
2. Adjust settings:
   - **WPM**: Start at 15-20, increase as you improve
   - **Frequency**: 700-800 Hz is most common
   - **Content Type**: Start with letters, then mixed
   - **Group Length**: Start with 1, increase for challenge
3. Click "Start Exercise"
4. Listen to the Morse code
5. Type your answer in the input box
6. Press Enter or click Submit
7. Get instant feedback:
   - Green border = Correct âœ“
   - Red border = Incorrect âœ—
8. Continue with the next exercise

**Tips**:
- Start slowly (10-15 WPM)
- Increase WPM by 2-3 as you master each level
- Write down difficult characters
- Practice regularly (daily is better than sporadic)

**Scoring System**:
- Real-time feedback on correct/wrong
- Cumulative score displayed
- Scores saved in your browser for history

**Common Challenges**:
- Too fast? Reduce WPM
- Sound too quiet? Check volume and frequency
- Getting frustrated? Take a break!

### Method 2: Koch Method - Structured Learning (â­ Recommended for Beginners)

**What it does**: Progressive learning system proven effective for Morse code

**Best for**:
- Complete beginners
- Systematic progression
- Building strong foundation
- Long-term learning (weeks/months)

**How it works**:
- Starts with 2 characters (K & M)
- Adds 2 more each lesson
- Eventually all letters and numbers
- 27 total lessons from A to Z + 0-9

**Three-Phase Learning Structure**:

#### Phase 1: Guided Listening (10-15 minutes)
- **Purpose**: Auditory and visual association
- **What happens**:
  1. You see a character (e.g., "K")
  2. Click "Play" to hear its Morse code
  3. Select the character from multiple choice
  4. Repeat 2 times for each new character
- **Goal**: Recognize the character's unique sound

#### Phase 2: Random Testing (10-15 minutes)
- **Purpose**: Recognition without visual hints
- **What happens**:
  1. Random character plays
  2. Click "Play" button to replay
  3. Select from available characters
  4. Complete 15 correct answers to proceed
- **Error handling**:
  - Wrong answer â†’ Audio replays â†’ Try again
  - Encourages accuracy over speed

#### Phase 3: Validation Exercises (15-20 minutes)
- **Purpose**: Confirm mastery of all lesson characters
- **What happens**:
  1. 30 random exercises from all lesson characters
  2. Mix of new and previously learned
  3. Same error handling as Phase 2
- **Completion**: Unlocks next lesson

**How to use**:
1. Click "Koch" on home page or navbar
2. Select "Lesson 1: K & M" to start
3. Follow the three phases in order
4. Complete all exercises
5. See celebration when lesson finishes
6. Choose "Next Lesson" or "Repeat This Lesson"
7. View completed lessons with checkmarks

**Sample Lesson Progression**:
```
Lesson 1:  K, M
Lesson 2:  K, M, R
Lesson 3:  K, M, R, S
Lesson 4:  K, M, R, S, U
...
Lesson 26: All 26 letters
Lesson 27: Numbers 0-9
```

**Tips for Success**:
1. **Don't skip phases**: Each phase builds the next
2. **One lesson per day**: Give your brain time to process
3. **Consistent practice**: 20-30 minutes daily is ideal
4. **Review previous lessons**: Use "Repeat This Lesson"
5. **Adjust WPM if stuck**: Slower helps mastery
6. **Track progress**: Lessons show completion status

**Expected Timeline**:
- Complete Beginner â†’ Lesson 10: ~2-3 weeks
- Lesson 10 â†’ Lesson 26: ~4-6 weeks
- Full Mastery (Lesson 27): ~6-8 weeks with daily practice

**Progress Tracking**:
- Green checkmark (âœ“) shows completed lessons
- Active lesson highlighted in blue
- Can repeat any lesson anytime
- Progress automatically saved

### Method 3: Passive Listening - Effortless Learning

**What it does**: Background learning through continuous character playback

**Best for**:
- Learning while multitasking
- Building rhythm recognition
- Relaxation and familiarity
- Supplementing active practice
- Waiting for other activities

**Listening Modes**:

1. **Random Letters** (A-Z)
   - Pure letter practice
   - No numbers or mixed content
   - Great for letter focus

2. **Random Numbers** (0-9)
   - Pure number practice
   - Useful after mastering letters
   - Less common in practice

3. **Mixed** (A-Z + 0-9)
   - Realistic mixed content
   - Most challenging
   - Best for real-world scenarios

4. **Groups of 5** (Pro Mode)
   - Groups of 5 random characters
   - Increases processing load
   - Improves retention

5. **Common Words** (Phrase Learning)
   - Pre-selected common English words
   - Natural rhythm learning
   - Better for conversation practice

**How to use**:
1. Click "Listen" on home page or navbar
2. Adjust settings:
   - **WPM**: 10-20 for passive learning
   - **Frequency**: Personal preference
   - **Mode**: Choose learning focus
   - **Speed**: 1 = slower, 5 = faster between characters
3. Click "Start Listening"
4. Characters play automatically
5. Each character displays and plays
6. Transcript scrolls in real-time
7. Click "Stop" when done

**Passive Settings Explained**:
- **WPM**: Similar to Practice mode
- **Frequency**: Tone pitch of audio (Hz)
- **Display Speed**: Gap between character plays
  - Speed 1: 2.5 sec between characters (relaxed)
  - Speed 3: 1.5 sec (normal)
  - Speed 5: 500ms (challenging)

**Tips**:
- Play while reading, coding, or working
- Start with slower speeds (1-2)
- Use at least 2-3 hours per week
- Combine with active practice
- Increase speed as comfortable

**Ideal Usage**:
- Morning wake-up session: 15-20 minutes
- Background during work: 30+ minutes
- Evening wind-down: 20-30 minutes
- Study breaks: 5-10 minutes

**Listening for Retention**:
- Don't force attention
- Let your brain absorb patterns
- Rhythm recognition improves automatically
- Most effective over weeks/months

## Common Questions & Answers

### Q: How long does it take to learn Morse?
**A**: 
- Basic alphabet: 3-4 weeks (daily practice)
- Conversational speed: 2-3 months
- High-speed (30+ WPM): 6-12 months

### Q: How much should I practice daily?
**A**: 
- Beginner: 20-30 minutes daily
- Intermediate: 30-45 minutes
- Advanced: 45+ minutes
- More consistent beats longer sessions

### Q: Should I use my eyes or ears?
**A**:
- Koch Method: Eyes + ears together
- Practice Mode: Ears only (after learning)
- Passive: Flexible (ears focused)

### Q: What WPM should I start with?
**A**:
- Absolute beginner: 10-12 WPM
- Comfortably learning: 15-18 WPM
- Advanced: 20-25 WPM
- Increase by 2-3 WPM when comfortable

### Q: Why Koch Method?
**A**:
- Proven effective since 1934
- Psychologically sound progression
- Regular review of previous characters
- Builds confidence gradually
- Optimal learning path

### Q: Can I switch between methods?
**A**: 
- Yes! Mix them for best results
- Koch Method: Main structured learning
- Practice Mode: Reinforce after Koch
- Passive: Background support

### Q: No sound playing?
**A**:
- Check browser volume (F12 â†’ Console)

- Try different browser
- Interact with page first (audio requirement)

### Q: Data not saving?

### Q: Mobile app available?

### Q: Can I export my progress?

## Learning Tips from Experts

### 1. **Form Good Habits Early**
- Learn proper characters from start
- Mental image of each character
- Avoid shortcuts
- Speed comes naturally

### 2. **Space Matters**
- Between dots/dashes within letter
- Between letters
- Between words
- Proper spacing = better recognition

### 3. **Variety is Important**
- Mix random letters vs. words
- Mix speeds
- Mix with other Morse learners
- Practice in different environments

### 4. **Consistency Trumps Duration**
- 15 minutes daily > 2 hours weekly
- Regular practice strengthens memory
- Short breaks help retention
- Daily habit builds fluency

### 5. **Passive + Active Balance**
- Active (Practice/Koch): 70% of time
- Passive (Listening): 30% of time
- Both needed for complete learning
- Different brain regions involved

### 6. **When Stuck, Slow Down**
- Reduce WPM by 5-10
- Take a break (1-2 days)
- Practice different content type
- Return refreshed

### 7. **Celebrate Progress**
- Notice improvements
- Track easier characters
- Acknowledge speed gains
- Enjoy the learning journey

## Troubleshooting

### Audio Issues

**Problem**: No sound at all
**Solutions**:
1. Check system volume (unmuted)
2. Check browser volume
3. Test another website
4. Try different browser
5. Check audio permissions

**Problem**: Sound very quiet
**Solutions**:
1. Increase system volume
2. Increase browser volume
3. Change frequency (try 800-900 Hz)
4. Check headphone connection

**Problem**: Crackling or distorted sound
**Solutions**:
1. Reduce volume
2. Change frequency
3. Try different WPM
4. Update browser

### Performance Issues

**Problem**: Slow or laggy
**Solutions**:
1. Close other browser tabs
2. Restart browser
3. Clear browser cache
4. Check system resources
5. Use latest browser version

**Problem**: Frequent freezing
**Solutions**:
1. Clear browser cache
2. Disable browser extensions
3. Update graphics drivers
4. Restart computer

### Data Issues

**Problem**: Progress not saving
**Solutions**:
1. Check localStorage is enabled
2. Try incognito mode
3. Check browser privacy settings
4. Clear browser cache
5. Try different browser

**Problem**: Want to clear data
**Solutions**:
1. Open DevTools (F12)
2. Application â†’ Local Storage
3. Find entries with "dotdashlearn_"
4. Delete individual or all entries
